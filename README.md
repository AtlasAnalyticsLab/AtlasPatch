# SAM2 Fine-Tuning
## Installations:

1. Create conda environment for sam2
2. Install SAM2 from their github repo: https://github.com/facebookresearch/sam2
3. Download the SAM2 checkpoints: https://github.com/facebookresearch/sam2 (Model Description).
Follow the instructions in their repository to obtain the checkpoints, and place them in the same directory as the training/testing scripts.
